# 6️⃣ 포켓몬 포획 (Catching Pokémon)

> 원문 페이지: p.68~71
>

포켓몬을 얻는 방법은 포획 외에도 여러 가지가 있습니다. 사람에게 자란 포켓몬과 야생 포켓몬 사이에는 큰 차이가 있으며, 야생 포켓몬은 인간에게 비협적이고 나타나면 달아나거나 공격적으로 마주합니다.

---

## 서식지 (Habitats)

포켓몬 세계는 넓고, 특정 포켓몬은 특정 지역에서만 발견됩니다. 연륜 있는 트레이너는 포켓볼을 들고 아무데나 들어다니지 않습니다.

---

## 특수 포켓몬

| 종류 | 설명 |
| --- | --- |
| **색다른 포켓몬 (Shiny)** | 부스러운 글리터 효과를 남김. 콘테스트에서 인기. 타입/능력치는 차이 없음 |
| **대체 폼 (Alternate Form)** | 같은 종이지만 다르게 생김. 타입/능력치/어빌리티 다를 수 있음 |
| **지역 필터 (Regional Variant)** | 특정 지역에서만 발견. 타입/어빌리티/진화 형태가 다를 수 있음 |
| **대형 포켓몬 (Overgrown)** | 일반보다 30~60% 크지만 매우 드물. **기본 HP +1** |

---

## 포켓볼 사용법 (Using Pokéballs)

포켓볼은 포켓마트에서 구매할 수 있습니다.

- 포획 시 칩에 트레이너 ID 등록 -> 타인 포켓볼로 포획 불가
- 포켓볼은 인간/나무/바위/자동차 등에는 사용 불가
- 한 번 사용된 포켓볼은 동일 내용물만 보관 가능

---

## 포획 롤 (Catch Roll)

| 아이템 | 실 효능력 |
| --- | --- |
| 포켓볼 | 4주사위 |
| 슈퍼볼 | 6주사위 |
| 울트라볼 | 8주사위 |

### 보너스 성공 수

| 야생 포켓몬 상태 | 추가 성공 |
| --- | --- |
| HP 절반 이하 | +1 |
| HP 1만 남음 | +1 |
| 스테이터스 이상 있음 | +1 |

> **기절한 포켓몬에는 보너스 성공 모두 상실**
>

### 랭크별 필요 성공

| 포켓몬 랭크 | 필요 성공 |
| --- | --- |
| 스타터 | 3 |
| 비기너 | 4 |
| 아마추어 | 6 |
| 에이스 | 8 |
| 프로페셔널 | 9 |

> 필요 성공에 미달하면 포켓볼은 파괴됩니다.
>

---

## 특수 포켓볼 (Special Pokéballs)

| 포켓볼 | 특징 |
| --- | --- |
| **프리미어 볼** | 신분의 상징. 다른 트레이너에게 세 보일 수 있음 |
| **럭셔리 볼** | 이것으로 포획하면 행복도 트레이트 향상 |
| **체리시 볼** | 리그와 연관된 자들에게만 지급 |
| **UB-볼** | 에테르 재단의 UB 프로젝트용 |
| **다이나맥스 볼** | 갈라르 과학의 자랑. 대형 홀로그램 생성 |
| **마스터 볼** | 실패 없이 모든 포켓몬 포획 (소문) |

---

## 포켓몬 얻는 다른 방법

| 방법 | 내용 |
| --- | --- |
| **교환** | 상대 트레이너 동의 후 포켓몬센터 기기에서 포켓볼끼리 교환. 교환 시 행복도/충성도 -1 (원래 포켓몬을 돌려받더라도 동일하게 적용) |
| **방생** | 포켓볼을 새 서식지에 놓아주거나 컴퓨터로 칩 해제 후 안전한 장소로 이송. 포켓몬에게 트라우마를 남기지 말 것. 단순한 이유로 방생하는 것은 리그 커뮤니티에서 눈총을 받습니다 |
| **구매** | 보육·훈련된 포켓몬. 비싸지만 성격, 특성, 특수기술 보장. 가격은 수십만~수백만 포켓달러. 저렴한 곳도 있으나 신뢰도는 보장 불가 |
| **입양** | 포켓몬 보호소에서 무료 입양. 등록된 모든 트레이너 가능. 버려진 포켓몬이 많아 경계심이 강할 수 있으나, 잘 돌보면 가장 충직한 동료가 됩니다 |
