# 8️⃣ 여행을 위한 아이템 (Items for the Journey)

> 원문 페이지: p.76~85
>

야생을 포함한 장거리 여행은 험합니다. 포켓몬 동료들이 안전을 지켜주지만, 편의는 직접 마련해야 합니다.

---

## 트레이너 기본 준비물

- **돈**: 시작 트레이너는 보통 $1,500 보유. 승리 시 포켓달러 획득, 패배 시 손실. 아르바이트로 추가 수입도 가능
- **식량**: 트레이너와 포켓몬 양쪽의 먹이. 직접 챙겨 주는 게 안전. 배고픈 야생 포켓몬에게 먹잇감이 될 수 있음
- **포켓몬 아이템**: 회복 아이템과 포켓볼 항상 휴대
- **캠핑 장비**: 동굴에는 위험한 야생 포켓몬이 들끓음. 텐트, 침낭, 도구를 챙길 것
- **이동수단**: 자전거, 포켓몬 탑승, 버스 등. 없으면 편한 신발이라도 챙길 것

## 회복 아이템

### 포션

스프레이 형태. **HP 1 회복 = 1유닛 소모 / 치명적 데미지 1 회복 = 2유닛 소모**

기절한 포켓몬에게 써도 상처를 안정시킬 수는 있지만 의식은 돌아오지 않습니다.

| 아이템 | 가격 | 효과 |

| 포션 | $400 | 2유닛 |
| 슈퍼 포션 | $700 | 4유닛. 골절에 가까운 상처도 회복 가능 |
| 하이퍼 포션 | $1,200 | 14유닛. 팀 전체에 나눠 쓰거나 중상에 한꺼번에 사용 가능 |
| 맥스 포션 | $1,700 | HP 완전 회복. 1회용 캡슐. 다음 날까지 추가 회복 불가 |

## 홀드 아이템 (Held Items)

포켓몬에 한 개의 홀드 아이템 슬롯. 전투에서 일부 효과가 발동됩니다. **홀드 아이템은 모두 미판매**, 모험 중 직접 구해야 합니다.

- 능력치를 변화시키는 홀드 아이템은 기술과 스택 불가 (높은 수치만 적용)
- 인간에게는 효과가 없음
- 한 씬(Scene)당 1개만 활성화 가능
