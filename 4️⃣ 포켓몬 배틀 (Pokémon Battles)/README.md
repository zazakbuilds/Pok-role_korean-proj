# 4️⃣ 포켓몬 배틀 (Pokémon Battles)

> 원문 페이지: p.42~62
> 

포켓몬 배틀은 이 세계의 주요 스포츠입니다. 신기한 회복 포션과 무료 의료로 동료들이 심각하게 다치는 일을 막아주기 때문입니다.

포켓몬 리그는 공식 경기를 최대한 안전하게 유지하려 노력합니다. 싸우는 선택은 당신에게 달렸지만, 포켓몬이 왜 싸우는지를 항상 기억하세요.
