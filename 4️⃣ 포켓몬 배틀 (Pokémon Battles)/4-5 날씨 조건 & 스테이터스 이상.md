# 4-5. 날씨 조건 & 스테이터스 이상

## 날씨 조건 (Weather Conditions)

포켓몬, 기술, 어빌리티는 날씨 조건에 의해 강화되거나 약화될 수 있습니다.

| 날씨 | 효과 |
| --- | --- |
| **맑음** | 불 속성 기술 데미지 +1 / 물 속성 기술 데미지 -1 / 얼음 상태 안 걸림 |
| **맹렬한 태양** | 불 속성 +1 / 물 속성 기술 실패 / 화상 상태 횟수 +2 / 얼음 상태 안 걸림 / 다른 날씨 활성화 불가 |
| **비** | 물 속성 +1 / 불 속성 -1 / 화상 회복에 필요 성공 -3 / 햇빛 강화 무시 |
| **폭우** | 물 속성 +1 / 불 속성 기술 실패 / 화상 상태 안 걸림 / 다른 날씨 활성화 불가 |
| **모래폭풍** | 매 라운드마다 바위/땅/강철 타입 제외 포켓몬이 1 데미지 입음 / 바위 타입 SDEF +1 |
| **강풍** | 비행 속성 +1 / 전기/얼음/바위 기술이 비행 타입에 효과 없음 / 진입 위험 무시 / 다른 날씨 활성화 불가 |
| **우박** | 매 라운드마다 얼음 타입 제외 포켓몬이 1 데미지 입음 / 얼음 타입 DEF +1 |

## 환경 조건 (Environmental Challenges)

공식 배틀 리그는 각 경기에 한두 가지 환경 조건을 추가할 수 있습니다.

| 환경 | 효과 |
| --- | --- |
| **안개/어둠** | 모든 포켓몬 기술 명중률 -1 |
| **진흙** | 이동 속도 절반 / 상대방과의 거리 진입 불가 |
| **화염 지대** | 매 라운드마다 모든 포켓몬이 화상 3 확률로 체크 |
| **전기 장** | 전기 속성 기술 데미지 +1 / 재충전 불필요 |

## 상태 이상 (Status Conditions)

상태 이상은 **동시에 여러 개** 적용될 수 있습니다 (예: 수면, 독, 화상 동시 가능).

Full Heal, Full Restore, 라즈베리만이 여러 상태 이상을 동시에 치료할 수 있습니다.

### 주요 상태 이상 목록

| 상태 이상 | 효과 | 지속 시간 | 면역 타입 |
| --- | --- | --- | --- |
| **화상 1단계** | 매 라운드마다 1 데미지 | DEX + ATK 4성공 누적 | 불 속성 |
| **화상 2단계** | 매 라운드마다 2 치명 데미지 | DEX + ATK 6성공 누적 | 불 속성 |
| **화상 3단계** | 매 라운드마다 3 치명 데미지 | DEX + ATK 8성공 누적 | 불 속성 |
| **마비** | DEX -2, 이동 속도 절반 | 24시간 | 전기 속성 |
| **동결** | 모든 행동 불가 / 얼음 껍질 (HP 5, DEF 2, SDEF 2) / 라운드마다 탈출 시도 | 녹을 때까지 | 얼음 속성 |
| **독** | 매 라운드마다 1 데미지 | 치료될 때까지 | 독, 강철 속성 |
| **맹독** | 매 라운드마다 1 치명 데미지 | 치료될 때까지 | 독, 강철 속성 |
| **수면** | 깨어나는 동안 모든 행동 불가 | 5분 | - |
| **혼란** | 모든 행동 성공 -1 / 실패 시 1 데미지 | 5라운드 또는 교체 | - |
| **풀죽음** | 다음 행동 손실 | 1행동 | - |
| **속박** | 이동 또는 특정 기술 사용 불가 | 5라운드~1시간 | - |
| **매력** | 상대방과 그 동료에게 절반 데미지만 부여 | 1시간 또는 교체 | 동성, 무성별 |

> **독/맹독**: 전투 밖에서도 지속적으로 데미지를 줍니다. 포켓볼 안에 있어도 마찬가지입니다!
