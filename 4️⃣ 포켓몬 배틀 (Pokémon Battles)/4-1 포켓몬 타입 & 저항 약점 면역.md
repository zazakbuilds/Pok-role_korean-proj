## 포켓몬의 다양한 타입

이 세계는 포켓몬이 제어할 수 있는 다양한 에너지 원체로 가득 차 있습니다. 포켓몬은 그것을 신체의 일부로 흡수하거나 강력한 공격으로 형성합니다.

특정 에너지 사용에 특히 능숙한 포켓몬에게는 **타입(Type)**이라는 레이블이 붙습니다. 일부 포켓몬은 두 가지 타입의 에너지를 사용할 수 있으며, 이 경우 두 번째 타입의 저항과 약점도 함께 적용됩니다.

지금까지 **18가지 타입**이 인정되었습니다.

| 타입 | 설명 |
| --- | --- |
| 노말 (Normal) | 다른 에너지 제어에 특별한 소질이 없는 포켓몬 |
| 벌레 (Bug) | 떼 지어서 힘을 발휘하는 곤충류 포켓몬 |
| 악 (Dark) | 나쁜 감정의 힘을 제어하며 불명예한 행동으로 유명 |
| 드래곤 (Dragon) | 내면의 분노로 돌진하는 전설의 생물 |
| 전기 (Electric) | 전류를 제어. 주변을 에너지로 획득 |
| 페어리 (Fairy) | 포착하기 어렵고 장난기 많은 생물. 기쁨과 눈물을 가져다줌 |
| 격투 (Fight) | 신체를 무기로 단련. 스스로의 에너지를 공격으로 구현 |
| 불꽃 (Fire) | 열에 강함. 불을 생성하고 형성하여 태움 |
| 비행 (Flying) | 바람 기류를 제어. 하늘의 지배자 |
| 고스트 (Ghost) | 지하세계에서 온 존재. 생명체의 생체 에너지를 먹이로 삼음 |
| 풀 (Grass) | 식물 같은 외모. 햇빛을 먹고 자람 |
| 땅 (Ground) | 땅 속에 삽니다. 대지의 움직임을 제어 |
| 얼음 (Ice) | 얼음과 눈이 신체를 이루었음. 극단적 저온에 강함 |
| 독 (Poison) | 독을 신체에 품고 다님. 어디든 병을 퍼뜨림 |
| 에스퍼 (Psychic) | 정신 에너지를 활용해 초능력을 사용. 지구에서 가장 똑똑한 생물 |
| 바위 (Rock) | 신체가 거대 암석. 낙석을 만들고 적을 분쇄 |
| 강철 (Steel) | 신체를 금속으로 덧입은 존재. 냉혹하고 기계적 |
| 물 (Water) | 수중에서 호흡 가능. 비를 부르고 강한 파도를 일으킴 |

## 저항, 약점, 면역

### 저항 (Resistances)

노말 타입을 제외한 모든 타입은 특정 기술 타입에 저항할 수 있습니다.

**저항**: 받는 데미지에서 **1을 차감** (눈에 띄지 않는 공격)

### 약점 (Weaknesses)

모든 포켓몬은 특정 기술 타입에 약점이 있습니다.

**약점**: 받는 데미지 **+1 추가** (단, 데미지 롤에서 최소 1 성공 필요)

### 면역 (Immunities)

일부 타입은 특정 타입의 데미지를 **전혀 받지 않습니다**. 단, 서포트 기술에는 여전히 영향될 수 있습니다.

### 이중 저항 / 이중 약점 (Double Resistance / Weakness)

- **두 타입이 모두 저항**할 경우: 저항 효과 **최대 -2 데미지**
- **두 타입이 모두 약점**일 경우: **+2 추가 데미지**
