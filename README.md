## 📖 프로젝트 소개 (About)
이 저장소는 한국 플레이어들이 **PokeRole** 시스템에 쉽게 입문할 수 있도록 핵심 규칙을 요약하여 제공합니다. 

수백 페이지 분량의 전체 룰북을 1:1로 번역한 것이 아니며, 실제 플레이에 즉시 적용 가능한 **핵심 메커니즘과 가이드**를 중심으로 구성되었습니다. 더 자세한 설정과 방대한 데이터는 [공식 홈페이지](https://www.pokeroleproject.com/)의 원본 룰북을 참고해 주시기 바랍니다.
