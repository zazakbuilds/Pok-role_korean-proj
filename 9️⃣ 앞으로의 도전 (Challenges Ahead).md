# 9️⃣ 앞으로의 도전 (Challenges Ahead)

> 원문 페이지: p.473~489
>

포켓몬 세계에서 트레이너를 기다리는 수많은 도전들입니다. 아래 내용은 에피소드식 이야기와 캠페인을 위한 아이디어와 틀을 제공합니다. 친구들과의 게임에서 영감으로 활용하세요.

---

## 세션 형식 선택

### 이번 주의 에피소드 (단발 세션)

한 세션 안에 완결되는 모험입니다. 포켓몬과 함께 어려운 사람을 돕는 가볍고 유쾌한 이야기, 동굴 탐험, 고된 트레이닝 세션, 특정 포켓몬을 찾아 도감을 완성하는 이야기 등을 담을 수 있습니다.

이 형식은 간단하고 입문자에게 적합합니다. 플레이어의 랭크에 맞는 조우를 직접 만들어 능력을 시험해보세요.

영감을 얻고 싶다면 짧은 우화나 포켓몬 TV 시리즈를 참고하세요. 에피소드를 그대로 재현하면서 친구들과 다른 결말을 만들어보는 것도 좋습니다!

> 모험의 모든 요소는 세션이 끝날 때 마무리되어야 합니다.
>

### 라이벌 / 악당 단체 격파

당신을 개인적으로 괴롭히는 사람들, 또는 혼란을 일으키는 범죄 조직을 상대하는 이야기입니다.

"Gary"와의 친선전일 수도 있고, 탐정과 함께 범죄 조직의 음모를 파헤쳐 단죄하는 이야기일 수도 있습니다.

### 포켓몬 리그 도전

리그는 트레이너에게 다양한 도전을 제공합니다:

**체육관 도전**: 초보 트레이너도 전략만 잘 짜면 체육관 관장에게 도전할 수 있고, 베테랑은 마지막 배지를 따내야 합니다. 다음 체육관까지의 여정 자체가 하나의 모험이 됩니다.

**연간 토너먼트 도전**: 1년 안에 배지 8개를 모으면 해당 지역 연간 토너먼트에 초대됩니다. 거기서 에이스 트레이너 칭호를 얻고, 지역 최강 포켓몬들과 맞붙습니다.

**챔피언 로드 도전**: 연간 토너먼트 상위 5위 안에 들면 챔피언 로드에 진입할 자격이 생깁니다.

### 포켓몬 세계의 전설

인간사에 지쳤다면, 소문과 전승을 따라 인간이 발을 들인 적 없는 세계의 끝으로 향하세요. 거기서 당신은 가늠할 수 없는 힘을 마주하게 됩니다.

메가진화, 다이맥스, 거다이맥스, Z기술로 모험의 수준을 높이고, 전설 포켓몬의 힘을 손에 넣으세요.

> 전설 포켓몬은 하나하나가 그 자체로 완전한 캠페인이 될 수 있습니다. 그들은 지하 던전의 최종 보스가 아니라, 뚜렷한 개성과 동기를 가진 존재입니다.
>

## 라이벌 시스템 (선택)

"당신이 강해질수록, 당신이 실패하길 바라는 사람이 많아진다." 라이벌은 그중 최악입니다.

라이벌은 항상 한 발 앞서 있으며, 당신의 길목에 나타나 아무리 노력해도 자기보다 못하다는 것을 상기시켜 줍니다.

**캐릭터 생성 시 라이벌도 함께 만드세요.**

### 라이벌 생성 규칙

- 태도(Attitude) 1개 선택
- 관계(Relationship) 1개 선택
- 배경(Background) 1개 선택
- 고유능력(Unlike-Ability) 1개 선택
- 비기너/아마추어/에이스/프로/마스터/챔피언 랭크마다 고유능력 1개씩 추가
- 라이벌의 포켓몬은 플레이어의 포켓몬에 **강한** 타입으로 구성
- **라이벌은 항상 플레이어보다 한 랭크 위**

> 라이벌에게는 능력치나 스킬 포인트가 필요 없습니다. 플레이어보다 주사위를 1~2개 더 굴린다고 가정하면 됩니다.
>

### 태도

| 태도 | 설명 |
| --- | --- |
| 덕스럽게 나쁜 놈 (Virtuous Jerk) | 겉으로는 정의롭지만 속은 최악 |
| 기만적 (Deceitful) | 항상 속임수를 씀 |
| 질투 (Envious) | 당신이 가진 모든 것을 탐냄 |
| 악 (Evil) | 순수하게 악함 |

### 관계

| 관계 | 설명 |
| --- | --- |
| 가족 | 가족 간의 라이벌 관계는 최악 |
| 친구 | 우호적인 경쟁 |
| 미스터리 | 이 사람은 누구? |
| 불리 | 자기 재미를 위해 괴롭힘 |
| 숙적 | 당신을 파괴하는 것이 인생의 목적 |

### 배경

| 배경 | 설명 |
| --- | --- |
| **핵포켓몬** | 합법적이지 않은 방법으로 포켓몬을 구함. 색다른 포켓몬에 희귀종만 보유 |
| **스타의 법칙** | 소셜 능력치가 최대. 항상 당신에 대한 거짓말을 하고 모두가 믿음 |
| **저주받은 축복** | 당신이 가장 원하는 것을 가지고 있지만, 그것을 저주로 여김 |
| **부잣집 도련님** | 무한한 돈을 갖고 있지만 $50 이상은 절대 안 걸음. 있는 마을마다 물가 3배 |
| **팀 멤버** | 범죄 조직 소속. 라운드 마지막에 자동으로 Beat Up 기술 사용 |
| **장애/고아** | 고난으로 가득 찬 삶. 당신이 "괴롭히는" 것처럼 보여 욕 먹음 |
| **안티히어로** | 당신이 어딜 가든 라이벌의 업적이 먼저. 무엇을 하든 라이벌이 먼저 더 잘함 |
| **폭군 스승** | 당신이 직면한 악당을 지원하고 아이템, 회복, 능력치 상승을 제공함 |

### 고유능력 (Unlike-Abilities)

| 능력 | 설명 |
| --- | --- |
| **반칙 면허** | 리그 규정을 어겨도 아무 처벌 없음. 당신이 어기면 중징계 |
| **먹고 들어감** | 데미지 롤이 항상 성공 |
| **과잉 자신감** | Will Point 무제한 |
| **참을 수 없음** | 대화할 때마다 Cool/Clever/Tough/Cute 관련 롤 자동 실패 |
| **남은 음식 중독** | 모든 포켓몬이 남은 음식 홀드 아이템 보유 (라운드당 HP 1 회복, 하루 최대 10) |
| **잔인한 트레이닝** | 모든 포켓몬이 최종 진화 상태 |
| **기술 장인** | 특정 강력한 기술 1개를 모든 포켓몬이 사용 가능. 중복 선택 가능 |
| **완벽한 존재** | 항상 5성공처럼 행동. 모든 기술에 절대 실패 없음 |
| **도둑 의심** | 만날 때마다 아이템이 사라짐. 포켓몬도 훔칠 수 있음 |
| **주인공 도둑** | 당신이 뭔가 대단한 일을 하면 라이벌이 모든 공을 가져감 |
| **은빛 혀** | 당신을 조종함. 믿는 대로 행동하게 만듦 |
| **불운의 기수** | 당신의 일상을 지옥으로 만듦. 자신감을 회복 불가, 매일 자신감 -2 |
| **이중 스파이** | 당신의 포켓몬 중 하나가 라이벌을 위해 일함. 중복 선택 가능 |
| **절대 못 이기는 상대** | 라이벌과 전투할 때 모든 주사위에서 성공 1개 차감 |

> **플레이어끼리는 라이벌 관계가 될 수 없습니다.** 고유능력은 밸런스가 잡혀 있지 않으며, 플레이어 캐릭터에게는 맞지 않습니다.
>

## 포켓몬 리그 정복

모든 트레이너의 꿈은 포켓몬 리그 챌린지에 한 번이라도 참가하는 것입니다. 지역 전체를 가로지르는 대규모 여정으로, 실력을 증명하는 여행입니다.

### 체육관 도전

리그에는 8개의 체육관이 있으며, 각각 강력한 체육관 관장이 이끕니다. 이기면 배지를 획득합니다. 배지는 엄청난 금액으로 구매할 수도 있습니다.

**1년 안에 배지 8개를 모아야 합니다.** 그렇지 않으면 다음 해부터 다시 시작해야 합니다.

### 연간 토너먼트

포켓몬 3마리로 교체 허용 단판 배틀. 경기장 테마가 다양합니다. 준결승은 6마리 단판, 결승은 매번 다른 형식으로 진행됩니다.

이 토너먼트에 참가한다는 것 자체가 큰 영광입니다. 이기든 지든, 보통 사람에게는 이것이 여정의 끝입니다.

### 챔피언 로드

연간 토너먼트 상위 5위에게만 허용되는 세계에서 가장 위험한 시험 장소입니다. 미로처럼 얽힌 던전으로, 도시만큼 넓고, 동굴처럼 어둡고, 전쟁터처럼 위험합니다.

포켓몬 마트도, 포켓몬 센터도 없이 며칠, 심지어 몇 주를 버텨야 합니다. 들어간 사람 대부분은 몇 시간 만에 포기합니다.

### 사천왕 도전

지역에서 가장 강한 트레이너 4명을 상대합니다. 이들은 모든 체육관을 정복하고, 챔피언 로드를 통과했으며, 사천왕을 한 명 이상 쓰러뜨린 자들입니다.

- 능력치 한계가 적용되지 않음
- 항상 포켓몬 6마리 보유
- 쉬지 않고 연속으로 4명을 모두 이겨야 함
- 사천왕 24마리의 포켓몬을 내 포켓몬 4마리로 처리하는 것이 이상적
- 이 전투는 방송되지 않으며 관중도 없음

### 챔피언 등극

챔피언은 전설적인 포켓몬과 같은 수준으로 행동합니다. 이들의 기술은 "극적 장치" 버전으로 간주될 만큼 강력합니다.

챔피언을 꺾으면 **챔피언 칭호**를 받고 **명예의 전당**에 영원히 기록됩니다.

> **챔피언, 마스터, 전설.** 이것은 포켓몬의 경이로운 세계 안에서의 첫 걸음일 뿐입니다.

## 메가진화

메가진화는 메가스톤에 담긴 힘을 포켓몬에게 주입하는 것입니다. 일반 진화와 유사하게 신체가 변화하고 능력치가 상승합니다.

메가진화가 끝나면 효과가 사라집니다. 메가진화는 포켓몬에게 폭력적인 과정일 수 있으므로, **트레이너와의 강한 유대감이 필수**입니다.

### 메가진화 규칙

- 포켓몬은 메가스톤을 홀드 아이템으로 보유, 트레이너는 키스톤 착용 필요
- **최종 진화 형태**만 메가진화 가능
- 메가진화 시 Will Point 1 소모
- 지속 시간: 한 씬 또는 한 배틀
- 스토리텔러 재량으로 포켓몬이 제어 불능이 될 수 있음

> 메가스톤은 각 포켓몬마다 존재하며, 매우 희귀합니다. 가문의 가보이거나 봉인된 유물인 경우가 많습니다.
>

## 영웅의 의식 (Z기술)

알로라 지방에서는 성인식으로 포켓몬과 대지와의 교감 여행을 떠납니다. 4개의 섬에는 각각 수호 전설 포켓몬이 있으며, 카후나의 안내로 시련을 통과하면 Z기술의 힘을 얻게 됩니다.

### Z기술 규칙

- 포켓몬은 Z크리스탈을 홀드 아이템으로 보유, 트레이너는 Z링 착용 필요
- Z기술 사용에는 특수 의식 댄스 필요
- 섬 시련 중 하나 이상을 통과해야 획득 가능
- 자세한 규칙은 p.428 참조

## 다이맥스 & 거다이맥스 (가라르 지방)

가라르 지방의 "파워 스팟"에서 포켓몬이 거대화하는 현상입니다. 일부 포켓몬은 크기만 커지는 것이 아니라 외형도 변하는데, 이것을 거다이맥스라고 합니다.

다이맥스/거다이맥스 상태에서는 **맥스 기술**과 **거다이맥스 기술**만 사용 가능합니다.

### 다이맥스 규칙

- 트레이너의 다이맥스 밴드 + 포켓몬 스타디움 내부 필요
- 야생의 파워 스팟에서는 밴드 없이도 발동 가능
- 전설 포켓몬 일부를 제외한 모든 포켓몬 사용 가능
- **다이맥스 효과**: 기본 HP +6, 맥스 기술 사용 가능 (p.425)
- 다이맥스 밴드 발동 시 지속 시간: **3라운드**
- 파워 스팟 발동 시: 무기한
- 다이맥스 포켓몬 면역: 풀죽음, 체중 기반 기술, 즉사 기술, 운명의 유대, 전장 이탈, 어빌리티 변경/제거
- 맥스/거다이맥스 기술은 복사 불가

### 거다이맥스 규칙

- 거다이맥스 인자 + 다이맥스 밴드 + 포켓몬 스타디움 필요
- 스토리텔러 재량으로 특정 포켓몬만 가능
- **거다이맥스 효과**: 기본 HP +12, 거다이맥스 기술 사용 가능 (p.425)
- 나머지 규칙은 다이맥스와 동일

> 같은 종 포켓몬이라도 거다이맥스 인자를 가진 개체와 그렇지 않은 개체가 있습니다.

## 신화와 전설 / 전설 포켓몬

Poké롤은 기본적으로 일상적인 모험을 다루지만, 이 세계에는 더 거대한 신화가 존재합니다.

### 전설 포켓몬 취급 원칙

- 그들의 존재는 주변 모든 것을 바꿈
- 대부분의 포켓볼로는 포획 불가
- 인간보다 지혜롭고 현명함
- 타입과 무관하게 어떤 기술도 사용 가능
- Z기술, 맥스 기술, 거다이맥스 기술 사용 및 의지대로 메가진화 가능
- 일부는 말을 할 수 있음 (텔레파시 또는 거친 목소리)
- 각각 단 하나만 존재 (일부 예외)

> **전설 포켓몬은 선악의 개념을 초월합니다.** 기껏해야 관심이 있거나 없거나 둘 중 하나입니다.
>

## 전설 랭크 시스템 (선택)

전설 랭크는 랭크 시스템의 확장판입니다. 아마추어 랭크이면서 동시에 영웅 랭크를 가질 수 있습니다.

**중요**: 트레이너는 트레이닝 세션으로 포켓몬에게 전설 랭크를 부여할 수 없습니다. 오직 전설 포켓몬만이 그럴 수 있으며, 자신의 철학에 맞는 뛰어난 포켓몬에게만 부여합니다.

| 랭크 | 설명 | 주요 혜택 |
| --- | --- | --- |
| **영웅 랭크** | 전설 포켓몬의 심부름꾼. 대부분 자신의 영웅 위치를 모름 | 능력치 최솟값 +1, 한계 +1, 기술 1개 습득, 영웅 어빌리티 |
| **수호자 랭크** | 인류에게 수호신으로 추앙받는 존재 | HP +20, 모든 능력치 최대화, Z기술 무제한, 어빌리티 3개 |
| **반신 랭크** | 세계를 움직이는 존재. 창조자는 아니지만 그에 준하는 힘 | - |
| **신 랭크** | 세계를 창조하고 대륙을 만들며 생사를 지배 | - |
| **최초의 자녀 랭크** | 창조주의 진정한 자식들. 물리 법칙과 신의 계율을 만든 존재 | - |
| **창조주 랭크** | 개념 자체이며, 상상력 외에는 어떤 한계도 없음 | - |

> **전설 랭크와 전설 포켓몬은 게임을 치명적인 여정으로 바꿉니다.** 높은 데미지, 높은 위험, 전 세계적 재앙. 이제 보조 바퀴를 떼어낼 시간입니다.
